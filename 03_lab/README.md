### Результат
Сделан многоступенчатый образ, который занимает ок. 60 мегабайт.
### Как запустить
1. Перейти в папку с Dockerfile
2. Ввести команду docker build -t docker-pt3 .
3. Ввести команду docker run -p 5000:5000 docker-pt3
